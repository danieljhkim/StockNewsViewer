## Getting Started
   
1. First clone the repo from git by running the following command:
    ```
    $ git clone https://github.com/realcryptogates/StockNewsViewer.git
    ```
2. Install all the dependencies. 
    ```
    $ npm install
    # or
    $ yarn add 
    ```
3. Install Expo.
    ```
    $ npm i -g expo
    # or
    $ yarn global add expo 
    ```
4. Start the application.
    ```
    $ npm start
    # or
    $ yarn start
    ```
